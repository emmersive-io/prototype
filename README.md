# ui-prototype

## install 
    $ curl https://install.meteor.com/ | sh
    $ git clone git@github.com:emmersive-io/ui-prototype.git
    $ cd ui-prototype
    $ meteor
    open browser http://localhost:3000
    
## deploy to digital ocean
    $ cd .deploy
    $ mup deploy
    open browser http://104.236.54.83:3000
